# Lebahhack

Lebahhack adalah proyek eksperimen digital yang berfokus pada pengembangan website modern, performa tinggi, dan optimasi SEO natural.

Project ini dibangun menggunakan Astro untuk menghasilkan website yang cepat, ringan, dan mudah di-deploy ke berbagai platform seperti Cloudflare Pages.

## Fokus Utama

- Pengembangan website modern
- Optimasi SEO natural
- Static site generation
- Cloudflare deployment
- Automasi workflow
- Eksperimen UI modern

## Kenapa Menggunakan Astro?

Astro memungkinkan website memiliki performa tinggi dengan JavaScript minimal. Hal ini membantu meningkatkan Core Web Vitals dan pengalaman pengguna.

Beberapa keuntungan Astro:

- Build cepat
- SEO friendly
- Struktur modular
- Mudah dipindahkan antar server
- Cocok untuk blog dan landing page

## Integrasi Cloudflare

Deploy menggunakan Cloudflare Pages memberikan beberapa keuntungan:

- CDN global
- SSL otomatis
- Deploy langsung dari GitHub
- Build otomatis
- Cache cepat
